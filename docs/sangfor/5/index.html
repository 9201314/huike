  外部启动会的目的

  1. 所有的合同内容是否一致
  2. 客户信息表、资产购买清单，服务内容服务计划
  3. 安全运营服务风险告知书，安全运营服务保密协议，授权函
  4. 项目背景信息（客户信息，客户对接人，客户关键人）
  5. 基本的网络环境（安全设备，拓扑，网段）
  6. 沟通的方式，服务方式等等
